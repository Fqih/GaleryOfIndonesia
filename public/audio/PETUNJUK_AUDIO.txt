=========================================================
PANDUAN MENAMBAHKAN AUDIO NARASI UNTUK INSPEKSI ARTEFAK
=========================================================

Folder ini (public/audio) adalah tempat Anda meletakkan semua aset suara untuk website.
Saat ini, folder ini sudah berisi musik latar (Sabilulungan.mp3).

UNTUK MENAMBAHKAN SUARA NARASI ARTEFAK:
1. Rekam atau siapkan file audio/suara penjelasan untuk masing-masing item.
2. Simpan file tersebut dalam format .mp3 untuk kompabilitas browser terbaik.
3. Beri nama file tersebut persis seperti berikut:
   - narasi1.mp3 (Untuk item pertama: Peta Indonesia)
   - narasi2.mp3 (Untuk item kedua)
   - narasi3.mp3 (Untuk item ketiga)
   - narasi4.mp3 (Untuk item keempat)
4. Masukkan (Copy-Paste) keempat file mp3 tersebut langsung ke dalam folder ini.

Jika nama file Anda berbeda (misalnya: narasi-peta.mp3), Anda harus membuka file 
`src/components/ArtifactInspector.jsx` dan mengubah bagian `audio:` pada variabel 
`artifactDetails` agar sesuai dengan nama file yang Anda buat.

TIPS OPTIMASI:
Usahakan ukuran masing-masing file mp3 di bawah 2 MB agar website Anda tetap ringan
saat diakses oleh pengunjung.
